# Cómo levantar el proyecto

Abre **dos terminales**:

### Terminal 1 — Backend
```bash
cd backend && node server.js
```
Corre en http://localhost:3001

### Terminal 2 — Frontend
```bash
cd frontend && npm run dev
```
Corre en http://localhost:5173
